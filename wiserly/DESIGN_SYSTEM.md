# Carbon Mint Design System

## Visual Identity

### Color Palette
- **True Charcoal** (#0B0D10) - Deep, matte background
- **Neo-Mint** (#A8E6CF) - Electric green for active states, CTAs, highlights
- **Neutral Gray** (#1E2125) - Secondary backgrounds
- **Off-White** (#F5F5F5) - Text on dark backgrounds

### Typography
- **Headings**: Bricolage Grotesque (quirky, personality-driven)
- **Body**: Inter (clean, readable)
- **Monospace**: JetBrains Mono (for code blocks)

## UX Principles

1. **Progressive Interface** - Simple for beginners, detailed code view for pros
2. **Dark Mode First** - Reduces eye strain for developers
3. **Action Cards** - Big, clear tiles instead of complex menus
4. **Observability** - Always show what the AI is doing in real-time

## Component Guidelines

### Buttons
- Primary (Neo-Mint): Active CTAs (#A8E6CF text on #0B0D10)
- Secondary (Gray): Less prominent actions
- Status indicators: ✅ Success, ⚠️ Warning, ❌ Error

### Cards
- Background: #1E2125
- Border: Subtle Neo-Mint glow on hover
- Spacing: 1.5rem padding

### Accessibility
- Contrast ratio: WCAG AA minimum (4.5:1)
- Focus states: Visible 2px Neo-Mint outline
- Keyboard navigation: Full support
