```bash

open wiserly/frontend/index.html

# Test login page
open wiserly/frontend/login.html
```

### 2. Deploy to Production ($0/month)
```bash
chmod +x deploy.sh
./deploy.sh
```

This will:
1. Deploy frontend to Cloudflare Pages (free, global CDN)
2. Deploy backend to Railway Hobby tier ($0-5)
3. Setup MongoDB free tier
4. Configure Stripe payments

```python
from quiqly_core.factory import QuiqlyFactory

factory = QuiqlyFactory()

# All 22 Master_Library modules copied instantly
```
